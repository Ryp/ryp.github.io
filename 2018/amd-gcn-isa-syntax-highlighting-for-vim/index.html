AMD GCN ISA Syntax Highlighting for Vim

June 29, 2018

I got into low-level shader optimizations lately, and realised that not many tools are provided for the job. Since i’m looking at GCN assembly regularly now, i used the occasion to write a decent syntax highlighting for it inside vim.

That way i got to go through the whole GCN ISA document (props to AMD for having the only public GPU ISA out there) as well as learning some vim scripting basics!

You can find the plugin on GitHub.

Writing a simple syntax plugin for vim was overall a pleasant experience, and i strongly recommend taking a look at AMD’s GCN3 ISA document if you care about performance (or just GPU architecture in general).
